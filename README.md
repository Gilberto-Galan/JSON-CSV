# JSON - CSV

## Proyecto

Este tiene la funcion de convertir JSON a CSV que atraves de los recuadros
ingresas el JSON y al dar click en convertir, lo transforma en CSV

Tiene que ser un arreglo para su conversion ejemplo:

```bash 
[
    {
      "id": 0,
      "name": "Logan",
      "age": 205
    },
    {
      "id": 1,
      "name": "Paul",
      "age": 22
    }
]
```
